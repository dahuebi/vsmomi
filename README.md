vsmomi
======

VMWare vSphere CLI.

[travis](https://travis-ci.org/dahuebi/vsmomi?branch=master)

[pyvmomi](https://github.com/vmware/pyvmomi) (VMware vSphere API Python Bindings).

### Example Usage

##### List Virtual Machines
```
vs ls
vs ls -l # long
vs ls -s # short
```

### Install
```
git clone https://github.com/dahuebi/vsmomi.git
cd vsmomi
# Linux
source rc.sh
# Windows
call rc.cmd
```
